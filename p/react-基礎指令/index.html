# 操作指令

# 建立本地項目

# 使用 Vite 

1npm create vite@latest my-project1

或是

1npm create vite

# 使用 create-react-app 

1npx create-react-app

# 進入專案資料夾 

1cd my-project1

# 安裝額外包(如React) 

1npm install

# 啟動開發預覽 server 

1npm run dev   # vite
2npm start     # create-react-app

# Udemy 教材啟用起手式 

1npm install
2npm run dev

# 使用 redux

# 安裝工具箱 

1npm install react-redux
2npm install @reduxjs/toolkit

# 安裝開發工具 

1Google 搜尋 Redux devtool
